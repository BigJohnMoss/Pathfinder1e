# Roadmap

## Milestone 1 — Foundation (included)
- Data contracts and validation
- Four representative classes
- Starter race, feats, spells, options
- Core progression engine
- Minimal web interface

## Milestone 2 — Complete core rules
- All core classes and races
- Skills, saves, BAB, HP, and feat slots
- Spell-list and spell-slot support
- Equipment and encumbrance

## Milestone 3 — Advanced character options
- Archetypes and compatibility checks
- Traits, domains, bloodlines, mysteries
- Prestige classes and multiclass validation
- Animal companions, familiars, eidolons

## Milestone 4 — Product features
- Character persistence
- Import/export
- Printable sheet and PDF output
- Shareable character links
